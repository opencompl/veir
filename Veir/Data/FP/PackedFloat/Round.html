    def Veir.Data.FP.PackedFloat.round {e s : Nat} (mode : RoundingMode) (etarget starget : Nat) (pf : PackedFloat e s) :
    PackedFloat etarget starget

    Round a PackedFloat e s into target format PackedFloat etarget starget using the given IEEE-754 rounding mode.

    The implementation is the composition PackedFloat → EDyadic → (rounded) EDyadic → PackedFloat:

    • PackedFloat.toEDyadic converts the input to its precise dyadic value (preserving NaN, ±∞, and ±0).
    • Dyadic.round mode rounds finite nonzero values to a representable value in (etarget, starget) per the given rounding mode.
    • EDyadic.pack re-encodes the result in the target packed format.

    NaN, ±∞, and ±0 are propagated through; only the exponent and significand widths change.
