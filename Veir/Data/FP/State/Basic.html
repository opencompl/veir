    inductive Veir.Data.FP.State :
    Type

    A plain enumeration of the possible states of a floating point value. We define this separately to enable bitblasting the state of a FP value, and to enable a shared representation of the FP state across packed/unpacked representations.

    • zero : State

      A zero floating point value.

    • subnormal : State

      A finite subnormal value.

    • normal : State

      A finite normal value.

    • infinite : State

      An infinite value.

    • nan : State

      A not a number (NaN) value.
