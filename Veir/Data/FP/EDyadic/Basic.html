    inductive Veir.Data.FP.EDyadic :
    Type

    An extended dyadic number (EDyadic) is a representation of a floating-point number that includes the special values zero, infinity, and NaN.

    • zero (sign : Bool) : EDyadic

      A zero with a sign bit.

    • nonzeroFinite (d : Dyadic) (hne : d ≠ 0) : EDyadic

      A nonzero finite number in dyadic notation. The proof hne : d ≠ 0 enforces that nonzeroFinite is never used to represent zero: signed zeros live in the zero constructor.

    • infinity (sign : Bool) : EDyadic

      An infinite number with a sign bit.

    • nan : EDyadic

      A NaN (not a number).

        Dyadic → EDyadic

        Smart constructor that maps an arbitrary Dyadic into an EDyadic, collapsing the Dyadic.zero case to EDyadic.zero sign. For nonzero dyadics, the result is .nonzeroFinite d _, with the d ≠ 0 proof discharged automatically.
