    structure Veir.Data.LLVM.Byte.Byte (w : Nat) :
    Type

    The Byte type can have any bitwidth w. It carries a two's complement integer value of width w and a per-bit poison mask modeling bitwise delayed undefined behavior.

    • val : BitVec w

      A two's complement integer value of width w.

    • poison : BitVec w

      A per-bit poison mask of width w.

    • h : self.val &&& self.poison = 0

      Invariant: if a bit is poison, the corresponding bit in val is zero.
