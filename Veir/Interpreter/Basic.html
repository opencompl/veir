    Veir Interpreter #

    This file contains a simple interpreter for a subset of the Veir IR.

    The interpreter maintains a mapping from IR values (ValuePtr) to runtime values (UInt64). Each supported operation reads its operands from this mapping and writes its results back into it.

    The interpreter walks the linked list of operations in a block. It continues until a func.return is encountered, at which point the returned values are collected and propagated to the caller.
