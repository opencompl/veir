    Equation Lemma and SSA Invariant #

    This file contains the definition of the equation lemma (EquationLemmaAt) and the proof of preservation when interpreting an operation.

    The equation lemma is based on the paper "A formally verified SSA-based middle-end" from Gilles Barthe, Delphine Demange, and David Pichardie. Our definition is slightly different as CompCertSSA semantics are based on small-step operational semantics.

    Equation Lemma #

    source
    def Veir.OperationPtr.Pure (op : OperationPtr) (ctx : IRContext OpCode) :
    Prop

    An operation is pure when its interpretation does not depend on, and does not modify, the memory state: running it under any memory yields the same result values and control flow, with the memory threaded through unchanged.

    Concretely, the result under memory₁ is the result under memory₂ with the output memory rewritten to the input memory.

      def Veir.InterpreterState.EquationHolds (state : InterpreterState) (ctx : WfIRContext OpCode) (op : OperationPtr) :
      Prop

      state.EquationHolds ctx op holds when state records the result of interpreting op. This is encoded by stating that interpreting op in state produces state itself, which is equivalent to saying that the results of interpreting op on the given state are present in state itself.

      Equations
      • state.EquationHolds ctx op = ∃ (controlFlow : Option Veir.ControlFlowAction), Veir.interpretOp ctx.raw op state = some (Veir.UBOr.ok (state, controlFlow))

        SSA Invariant at a Program Point #

        source
        def Veir.InterpreterState.EquationLemmaAt (state : InterpreterState) (ctx : WfIRContext OpCode) (location : InsertPoint) (_locInBounds : location.InBounds ctx.raw := by grind) :
        Prop

        An interpreter state satisfies the SSA invariant at a program point if it satisfies the equation lemma for all operations that dominate that program point. In other words, all operations that dominate the given location have already been interpreted and their results are present in the state.

        Equations
        • state.EquationLemmaAt ctx location _locInBounds = ∀ (op : Veir.OperationPtr), op.InBounds ctx.raw → op.Pure ctx.raw → op.dominatesIp location ctx → state.EquationHolds ctx op
