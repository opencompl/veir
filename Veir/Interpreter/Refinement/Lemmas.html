    Inversion #

    Inversion lemmas for RuntimeValue.isRefinedBy: given a refinement hypothesis v ⊒ tv where the source value v has a known constructor, these lemmas recover the shape of the target value tv.

    source
    theorem Veir.RuntimeValue.int_of_isRefinedBy {bw : Nat} {v : Data.LLVM.Int bw} {tv : RuntimeValue} (h : int bw v ⊒ tv) :
    ∃ (t : Data.LLVM.Int bw), tv = int bw t ∧ v ⊒ t

    A runtime value tv that refines an integer runtime value v is itself an integer of the same width, and the underlying integer value refines v.

    source
    theorem Veir.RuntimeValue.byte_of_isRefinedBy {bw : Nat} {v : Data.LLVM.Byte bw} {tv : RuntimeValue} (h : byte bw v ⊒ tv) :
    ∃ (t : Data.LLVM.Byte bw), tv = byte bw t ∧ v ⊒ t

    A runtime value tv that refines a byte runtime value v is itself a byte of the same width, and the underlying byte value refines v.

    source
    theorem Veir.RuntimeValue.float_of_isRefinedBy {bw : Nat} {v : Float} {tv : RuntimeValue} (h : float bw v ⊒ tv) :
    tv = float bw v

    A runtime value tv that refines a float runtime value v is equal to it.
