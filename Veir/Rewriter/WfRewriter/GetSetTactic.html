    @[simp_getset] attribute and simp_getset tactics #

    This module defines the simp_getset and simp_getset? tactics, which tries to apply all @[simp_getset]-tagged theorems. These theorems are equalities with a single explicit argument of the form f ... = .... The tactic then looks for hypotheses matching this shape, and applies the theorems with these hypothesese passed as arguments.
