    Rewriter.setType #

    Rewriter.setType only updates the type field of a value (an op result or a block argument). It touches no indices, owners, use links, parents, or def-use/op/block chains, and IRContext.WellFormed imposes no constraint on the type field, so it preserves well-formedness. Every chain is therefore "unchanged" by the rewrite.
