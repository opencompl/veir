              inductive Veir.Parser.Lexer.TokenKind :
              Type

              Token kinds that can be produced by the lexer.

              • eof : TokenKind

                End of file token

              • bareIdent : TokenKind

                Identifier

                bare-id ::= (letter|[]) (letter|digit|[$.])*

              • atIdent : TokenKind

                Identifier prefixed with an @ symbol.

                at-ident ::= @ (bare-id | string-literal)

              • hashIdent : TokenKind

                Identifier prefixed with a # symbol. ident only includes the identifier itself, not the # symbol.

                hash-ident ::= # bare-id

              • percentIdent : TokenKind

                Identifier prefixed with a % symbol.

                percent-ident ::= % bare-id

              • caretIdent : TokenKind

                Identifier prefixed with a ^ symbol.

                percent-ident ::= ^ bare-id

              • exclamationIdent : TokenKind

                Identifier suffixed with a ! symbol.

                exclamation-ident ::= bare-id !

              • floatLit : TokenKind

                Floating point literal.

                Integer literal.
