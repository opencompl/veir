    def Veir.matchOp (op : OperationPtr) (ctx : IRContext OpCode) (opType : OpCode) (numOperands : Nat) :
    Option (Array ValuePtr × propertiesOf opType)

    Match an operation that has a single result, a specific opcode, and a specific number of operands. Returns the operands and the properties of the operation if it matches, or none otherwise.
