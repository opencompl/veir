        Implement this class to register a custom type to be recognized as a fact type by the dataflow framework.

              structure Veir.DataFlowAnalysis :
              Type

              A single transfer problem scheduled by the fixpoint solver.

              • kind : AnalysisKind

                Tag to identify the implemented analysis.

              • init : OperationPtr → DataFlowContext → IRContext OpCode → DataFlowContext

                Given the top level operation pointer, initializes the analysis to a valid state. This often involves enqueueing some number of work items into the work list, such as every SSA value reachable from the top level operation pointer.

              • visit : InsertPoint → DataFlowContext → IRContext OpCode → DataFlowContext

                The transfer function, visiting the given InsertPoint.
