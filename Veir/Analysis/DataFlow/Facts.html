    Facts in the dataflow framework #

    In the mathematics of monotone dataflow frameworks, an analysis assigns abstract states (each from some lattice) to program points and repeatedly applies monotone transfer functions until a fixpoint is reached. A fixpoint f is such that f(x) = x. So in other words, when nothing changes.

    This file defines the objects used to store those abstract states. A Fact is one piece of analysis information and a LatticeAnchor says where in the IR that information lives (such as an SSA value or a CFG edge).
