    structure Veir.ValuePtr.DefUse {OpInfo : Type} [HasOpInfo OpInfo] (value : ValuePtr) (ctx : IRContext OpInfo) (array : Array OpOperandPtr) (missingUses : Std.ExtHashSet OpOperandPtr := ∅) :
    Prop

    A def-use chain for an SSA value. The def-use chain is represented as an ordered array of operands, where each operand corresponds to a use of the value. The first element of the array is the first use of the value. Each operand in the array points to the next use of the value, forming a linked list.

        structure Veir.BlockPtr.OpChain {OpInfo : Type} [HasOpInfo OpInfo] (block : BlockPtr) (ctx : IRContext OpInfo) (array : Array OperationPtr) (missingOps : Std.ExtHashSet OperationPtr := ∅) :
        Prop

        An operation chain owned by a block. An operation chain is a doubly linked list of operations within a block, where each operation points to the next and previous operations in the block. The block itself points to the first and last operations in the chain. The operation chain is represented as an ordered array of operation pointers, where the first element of the array is the first operation in the block, and the last element is the last operation in the block. Each operation that has the block as its parent must be included in the operation chain, unless it is included in the missingOps set.
